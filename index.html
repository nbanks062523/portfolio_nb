Project 2


In this project, Airflow was used to orchestrate a workflow that did the following:

  • Create a DAG (Directed Acyclic Graph) that executes tasks to add CSV files to Google Cloud beginning with two functions that act as a checklist to ensure that the data files are properly placed and to ensure that the DAG is able to connect to the Big Query client
  • The first 2 tasks of the DAG run the checklist functions, followed by an empty operator for branching the table creation tasks
  • The next task runs a loop that loads each table
  • Finally, another empty operator is used to bring the tasks back together


As shown in the images, the following tables were created by the DAG:

  • Food Inflation by month - A table that shows the inflation rates of food by month year over year
  • Grocery Prices by month - A table that shows the price of groceries month over month by state
  • SNAP Population, SNAP Poverty Population, SNAP Program Participants - These tables display data around the Supplemental Nutrition Assistance Program and its recipients over time by state


Languages,Tools and Skills Used

  • DAG Python Operators
  • Visual Studio Code
  • Airflow
  • Google Big Query
